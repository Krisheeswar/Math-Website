This is where I'll show you how 2=1


Consider a = b, Then a2 = ab

Subtract b2 from both sides

Then, a2-b2 = ab - b2

(a+b)(a-b) = b(a-b)   [We know that a2-b2 = (a+b)(a-b)]

(a+b)  (a-b) = b  (a-b)

a+b = b

b+b = b   [since a = b]

2b = b

2 b = b

2 = 1

Hence Proved
